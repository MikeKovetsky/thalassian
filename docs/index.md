# Thalassian Language Construction Package

Welcome to the documentation for the Thalassian Language Construction Package.

For the latest features and a quick-start guide, please see our [README](https://github.com/Portal-OS/thalassian#readme).

## Overview

The Thalassian package provides tools to translate, transcribe, and analyze the Thalassian language as spoken by the High Elves and Blood Elves of Azeroth. 

### Contents

- **[Citations](citations.md):** Canonical sources and references used to build the Thalassian dictionary and grammatical rules.
- **[Methodology - Grammar](methodology/grammar.md):** Documentation of the grammatical rules.
- **[Methodology - Lexicon](methodology/lexicon.md):** Details on vocabulary construction.
- **[Methodology - Typography](methodology/typography.md):** Information on the script and typography.

Feel free to explore the modules to learn more about the linguistic structure and how to use the translation pipeline.
